
# TriSteamOwl

3連1点バーストライフルです
武器を持って`attack`でリロード(リソース必要なし)
マグマバケツも持った状態で、武器を持って`sneak`+`attack`でマグマをチャージします

`use`を長押しすると射撃を行えます
射撃時にマグマと弾を消費します

## レシピ

レシピは以下の通り

```json
"pattern": [
    "ABA",
    "BCD",
    "ADE"
],
"key": {
    "A": "#minecraft:logs",
    "B": "copper_ingot",
    "C": "clock",
    "D": "water_bucket",
    "E": "barrel"
}
```
